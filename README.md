# 🌸 Iris Dataset Exploration & Visualization

## 🎯 Task Objective
The objective of this task is to explore and visualize the Iris dataset to understand data patterns, distributions, and relationships between features using basic data analysis techniques.

---

## 📊 Dataset Used
- Dataset: Iris Dataset  
- Source: Built-in dataset (Seaborn / Scikit-learn)  
- Type: Multiclass classification dataset  

It contains measurements of iris flowers including sepal length, sepal width, petal length, and petal width for three species.

---

## 🤖 Models / Techniques Applied
This task focuses on Exploratory Data Analysis (EDA) rather than machine learning models:

- Data loading and inspection
- Statistical summary
- Data visualization using:
  - Scatter plots  
  - Pair plots  
  - Histograms  
  - Correlation heatmap  

---

## 📈 Key Results and Findings
- Petal length and petal width are the most important features for distinguishing species  
- Iris Setosa is clearly separable from other species  
- Versicolor and Virginica show slight overlap  
- Strong correlations exist between petal features
